jq-metro (preview)

about jq-metro

jq-metro is a jquery plugin project started on April 2011 and aims to provide metro-style user interface for browsers via jquery.

  • project's home page

This page is a preview of a set of new features that are recently developed for jq-metro. Please note that these features have not yet been added to the library.

Use your mouse button, your device's touch or your keyboard arrow keys to swipe left or right to the other parts of this page and learn more about these new features.

Mohammad Valipour, September 2011
